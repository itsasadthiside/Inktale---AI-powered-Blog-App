# Aivory Blog 🖋️
A minimalistic AI-powered blog app for short, snappy posts.

## Features
- ✨ AI-generated content
- 🌓 Minimal, Medium-style UI
- 📸 Image uploads
- 💬 Comment system

## Stack
- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Run Locally
```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm start
